DEMO RÁPIDA (15 minutos):

1. Clonar repositorio:
   git clone https://github.com/GasparGW/rag-bpg-text-mining.git
   cd rag-bpg-text-mining

2. Instalar dependencias:
   python3 -m venv venv
   source venv/bin/activate  # En Windows: venv\Scripts\activate
   pip install -r requirements.txt

3. Ver análisis (notebooks):
   jupyter lab notebooks/
   # Orden: 01_data_exploration → 02_rag_optimization → 03_evaluation

4. (OPCIONAL) Probar aplicación funcional:
   # Requiere Ollama instalado: https://ollama.ai
   ollama pull llama3.1:8b
   ollama pull nomic-embed-text
   
   # Terminal 1 - API:
   python3 -m uvicorn api.main:app --port 8000
   
   # Terminal 2 - PWA:
   python3 -m http.server 8080 --directory pwa
   
   # Abrir: http://localhost:8080
